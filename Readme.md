# Web Programming – AUCA

This repository contains labs, and final projects completed for the Web Programming course at the American University of Central Asia.  
It serves as a record of learning progress and practice with modern web development technologies.

---

## Technologies Used

- **HTML5** — page structure  
- **CSS3** — styling and layouts  
- **JavaScript** — interactive behavior   
- **Git & GitHub** — version control
